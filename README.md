# STM32 四轮智能小车项目
## 项目简介
本工程是基于STM32单片机的智能小车完整源码，包含电机驱动、红外循迹、底层延时驱动等全套代码，可在Keil MDK中直接编译烧录。

## 工程目录说明
- hardware：硬件外设驱动（电机、传感器）
- user：小车上层业务逻辑
- system：系统底层工具、延时函数
- start：STM32启动文件
- Library：STM32标准外设库

## 编译环境
- 软件：Keil MDK5
- 主控芯片：STM32F1系列

## 使用方法
1. 将代码完整克隆到本地
2. 打开 `小车完整代码.uvprojx` 工程文件
3. 连接开发板，编译后下载程序即可运行
